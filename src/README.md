# Blackjack vite

Pasos para ejecutar el proyecto :

1. Clonar el repositorio 
2. Ejecutar ```nmp install``` Para construir los módulos node
3. Correr el ```nmp run dev``` 
4. Abrir la URL de localhost ``https:\\----localhost``

## Producción

1. Ejecutar el comando ```npm run build```
2. Tomar la carpeta ```dist``` y desplegarla
